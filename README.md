# Note Service
A service for creating your own notes
## Idea
The main idea is to create a simple application where you can create your own notes and like them.
# Technological stack
- SpringBoot as a skeleton framework
- MongoDB database
- Spring Web
- Spring Security
- Spring Data MongoDB
- Spring Boot Maven Plugin
- Lombok
- MapStruct
